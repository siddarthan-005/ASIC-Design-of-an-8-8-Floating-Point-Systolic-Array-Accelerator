# ASIC-Design-of-an-8-8-Floating-Point-Systolic-Array-Accelerator
This project focuses on the design and ASIC implementation of an 8×8 floating-point systolic array accelerator using a complete RTL-to-GDSII VLSI design flow.The design is implemented in Verilog HDL and verified through simulation. The verified RTL is then passed through a full ASIC design flow
